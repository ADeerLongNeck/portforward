# OPTIMUS UI 优化规格

## 问题分析
- 窗口 1200x800 过大，内容紧凑后留白过多
- 缺乏动画和交互反馈
- 布局不够精致

## 优化方案

### 1. 窗口配置
- 默认: 900x650
- 最小: 700x500

### 2. 动画效果
- 模式切换: 0.3s ease
- 状态卡片: 淡入 + 上移 0.2s
- 按钮: hover scale(1.02) 0.15s
- 表格行: hover 背景色渐变 0.2s
- 折叠面板: height transition 0.3s ease

### 3. 折叠面板
- 每个section可折叠
- 标题栏显示展开/收起图标
- 保存展开状态

### 4. 布局优化
- 减少不必要的间距
- 使用flexbox优化对齐
- 状态卡片改为4列（紧凑时2列）
